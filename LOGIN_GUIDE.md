## 🚨 **COMMON ISSUES & SOLUTIONS**

### **Issue 1: Login Nahi Ho Raha**
```
Solutions:
✅ Email correct check karo (no spaces)
✅ Password case-sensitive hai
✅ Clear browser cache
✅ Try incognito mode
✅ Check internet connection
```

### **Issue 2: Dashboard Nahi Khula**
```
Solutions:
✅ Wait 2-3 seconds after login
✅ Manually go to:
   /admin-dashboard
   /seller-dashboard
   /profile
✅ Refresh page
```

### **Issue 3: Logout Nahi Ho Raha**
```
Solutions:
✅ Click profile icon → Logout
✅ Or directly visit /auth
✅ Clear cookies manually
```

### **Issue 4: Images Upload Nahi Ho Rahe**
```
Solutions:
✅ Max 5MB per image
✅ JPG/PNG format only
✅ At least 2 product images needed
✅ 1 bill image mandatory
```
